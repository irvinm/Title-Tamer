Icon Coloring

If you want the addon icon to adapt to your theme exactly, you need to set a specific preference in Firefox.

What is this preference? By default, Firefox treats extension icons as static images. Enabling this setting allows the browser to pass theme data (like text color) to the icon. This lets Title Tamer dynamically adjust its icon color to match your toolbar, ensuring it stays visible on light, dark, or colorful themes.

Please follow these steps:

  1. Type about:config in your Firefox address bar and press Enter.
  2. Accept the risk warning if prompted.
  3. Search for svg.context-properties.content.enabled.
  4. Double-click the preference or click the toggle button to set its value to true.

Note: If you do not enable this setting, the addon icon will remain black regardless of your theme. It will not support being able to have a light or dark version.
